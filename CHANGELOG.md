## 0.0.1

- Initial release.
- Android, iOS, and macOS support via gomobile bindings.
- Exposed Xray core APIs: run, stop, test, ping, stats, geo, share links, DNS, and TUN fd.
- Added example app with config JSON input and start/stop controls.
- Added build script `scripts/build_and_wire.py` to compile and wire native artifacts.
